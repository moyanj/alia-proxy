# 仪表盘与监控

`aiprox` 配备了一个基于 Vue 3 构建的现代化 Web 仪表盘，为您提供全方位的系统监控、日志审计和模型调试能力。

通过访问 `http://localhost:3000` (默认端口)，您可以立即获得系统的可视化概览。

## 核心模块

仪表盘主要包含以下几个功能模块：

1.  **概览 (Dashboard)**: 系统的“驾驶舱”，提供关键指标的实时快照。
2.  **用量分析 (Analytics)**: 提供深入的数据分析图表，帮助您理解流量模式、成本分布和性能瓶颈。
3.  **日志管理 (Logs)**: 强大的日志查询和详情查看工具，支持多模态内容展示和 ShareGPT 导出。
4.  **演练场 (Playground)**: 一个交互式的模型测试环境，支持流式对话、参数调整和 JSON 模式。
5.  **配置查看 (Providers & Mappings)**: 展示当前系统加载的提供商和路由映射规则（只读）。

## 访问与鉴权

目前，仪表盘主要设计用于内部网络访问，默认不开启强制登录。在生产环境部署时，建议配合 Nginx 或其他网关进行 Basic Auth 或 IP 白名单限制，以确保安全。
